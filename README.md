# Controle de Estoque

## Tecnologias Utilizadas
- PHP 7.4
- Laravel 7
- MySQL 5.7
- Vue 2.5
- TailwindCSS 1.7

## Ambiente Local
### Pré-requisitos
- Docker
- Docker Compose
- Portas `8080`, `9000` e `3306` sem nada rodando

### Instalação
> Instala dependências e prepara o banco de dados
```
$ sh ./install.sh
```

### Rodando a aplicação
> Baixa containers e sobe a aplicação
```
$ sh ./run.sh
```

### Acesso ao sistema
URL: http://localhost:8080
- Login: `admin@admin.com`
- Senha: `admin`
